# LeanScale Team Scorecard

## Overview

Employee recruitment and performance review system with two components:
1. **Technical Skills** — 6 categories × 2 tracks (Pipeline Ops + Revenue Ops)
2. **Soft Skills** — 5 consulting competencies

## Scoring Scale

All scores are 1-5:
- **1** - Novice/Developing
- **2** - Competent/Building
- **3** - Proficient/Reliable
- **4** - Advanced/Excels
- **5** - Expert/Exceptional

## File Structure

## Technical Skills Categories

| Category | Pipeline Ops | Revenue Ops |
|----------|-------------|-------------|
| Planning | Marketing budgets, channel quotas, ICP | Capacity plans, territories, sales quotas |
| Process | Lead lifecycle, routing, attribution | Sales lifecycle, handoffs, comp plans |
| Systems | MAP, ABM, email automation | CRM, CPQ, revenue intelligence |
| Reporting | Marketing dashboards, pipeline metrics | Sales dashboards, forecasting |
| Enablement | Marketing training, content | Sales training, playbooks |
| People | Marketing/partner stakeholders | Sales/CS stakeholders |

## Soft Skills Categories

1. **Communication** — Written, verbal, client-facing
2. **Organization** — Task management, documentation, deadlines
3. **Diligence** — Attention to detail, quality, thoroughness
4. **Leadership** — Initiative, ownership, team development
5. **Value Alignment** — Cultural fit, integrity, client-first mindset

## Usage

### For Performance Reviews
1. Review employee's work over the period
2. Score each category 1-5 using the rubrics
3. Calculate averages for Technical, Soft Skills, and Overall
4. Discuss with employee and set development goals

### For Recruitment
1. Use rubrics to structure interview questions
2. Score candidate based on interview responses
3. Compare against role requirements
4. Use gaps to identify training needs for new hires

## Role Expectations

| Role | Technical Target | Soft Skills Target |
|------|------------------|-------------------|
| Engineer | 2-4 across categories | 3+ in core areas |
| Architect | 4-5 in specialty, 3+ elsewhere | 4+ across all |
