# Technical Skills Rubrics

## Scale Overview
- **1 - Novice**: Learning fundamentals, needs guidance
- **2 - Developing**: Can execute with support, building competence
- **3 - Proficient**: Works independently, solid execution
- **4 - Advanced**: Leads initiatives, mentors others
- **5 - Expert**: Industry-level mastery, innovates methodology

---
